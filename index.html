¿QUIEN ES ESE POKÉMON?

Vue.js | Axios

Aplicación interactiva que consume la PokéAPI para mostrar 20 pokémons con un filtro visual. Los usuarios deben adivinar su nombre; si aciertan, el pokémon se revela y se incrementa un contador. El proyecto implementa componentes con props, renderizado dinámico (v-for), eventos (métodos, ciclo de vida), comunicación hijo‑padre, bindings condicionales y propiedades computadas.

Link
Don-run

Vue 3 | Vite | HTML5 Canvas

Mini juego interactivo basado en el clásico Snake. Renderizado con canvas y controlado por teclado, el jugador guía a una serpiente personalizada para recolectar objetos sin chocar. Usa sprites personalizados para la cabeza, el rastro y los puntos. Implementa lógica de movimiento, detección de colisiones, sistema de puntaje, estados reactivos, y adaptación responsiva al tamaño de pantalla usando la Composition API.
